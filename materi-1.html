Pengenalan Dasar HTML untuk Pemula


1. Pengertian HTML

HTML adalah singkatan dari HyperText Markup Language , yaitu bahasa markah yang digunakan untuk membuat dan menyusun halaman web. Kata HyperText berarti teks yang memiliki tautan (link) ke halaman lain di internet. Sedangkan Markup Language berarti bahasa yang menggunakan tanda atau simbol (markup) untuk memberi struktur pada sebuah dokumen. HTML bukan bahasa pemrograman, melainkan bahasa yang berfungsi untuk memberi struktur pada konten agar dapat ditampilkan di browser.

Dengan HTML, kita bisa menampilkan teks, gambar, tautan, tabel, video, dan berbagai elemen lain. Secara sederhana, HTML dapat diibaratkan sebagai pondasi atau kerangka sebuah rumah web. Ia menjadi dasar dari semua konten di internet, sementara CSS berperan untuk mempercantik tampilannya, dan JavaScript menambahkan interaktivitas agar situs terasa hidup.

2. Sejarah Singkat Perkembangan HTML

HTML diciptakan oleh Tim Berners-Lee pada tahun 1989 di CERN, Swiss. Tujuannya sederhana adalah untuk memudahkan para ilmuwan berbagi dokumen dan menghubungkannya satu sama lain menggunakan link.

Sejak saat itu, HTML berkembang melalui beberapa versi:

  • HTML 1.0 (1991) – versi awal, sangat sederhana.
  • HTML 2.0 (1995) – mulai distandardisasi, menambah dukungan form dan tabel.
  • HTML 3.2 (1997) – menambah tag untuk tampilan seperti warna dan font.
  • HTML 4.01 (1999) – lebih rapi, mulai dipisahkan dari tampilan (mulai butuh CSS).
  • XHTML (2000-an) – gabungan HTML dan XML, lebih ketat aturan penulisannya.
  • HTML5 (2014-sekarang) – versi modern yang mendukung multimedia (video, audio), canvas untuk animasi, dan elemen semantik seperti <header>, <footer>, <article>, dan <section>.

3. Alat yang Dibutuhkan untuk Belajar HTML

Untuk belajar HTML, tidak dibutuhkan alat yang rumit. Cukup dua hal saja:

  1. Teks Editor

    2. Digunakan untuk menulis kode HTML.
    Contohnya:

    • Notepad (Windows)
    • Notepad++
    • Visual Studio Code (direkomendasikan karena lengkap dan ringan)
    • Sublime Text
  2. Web Browser

    3. Digunakan untuk melihat hasil tampilan HTML.
    Contoh browser: Chrome, Edge, Firefox, Opera, dan Safari.

Kamu cukup menulis kode HTML di teks editor, lalu menyimpannya dengan ekstensi <.html>, dan membukanya di browser untuk melihat hasilnya. HTML dapat dijalankan langsung oleh browser, jadi kamu tidak memerlukan koneksi internet atau software tambahan apa pun untuk mencobanya.

4. Struktur Dasar Dokumen HTML

Setiap halaman HTML memiliki struktur utama seperti ini:

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Belajar HTML #01</title>
  </head>
  <body>
    <p>Hello World!</p>
  </body>
</html>

Penjelasan bagian-bagiannya:

Bagian Fungsi
<!DOCTYPE html> Memberi tahu browser bahwa dokumen ini menggunakan HTML5.
<html> Elemen utama yang membungkus seluruh isi halaman.
<lang="en"> Menandakan bahasa halaman (bisa diganti “id” untuk bahasa Indonesia).
<head> Bagian kepala, berisi informasi meta, judul halaman, dan tautan ke file lain (seperti CSS atau JavaScript).
<title> Judul halaman yang muncul di tab browser.
<body> Bagian tubuh, tempat semua konten yang terlihat oleh pengguna ditampilkan.

5. Tag dan Elemen Dasar HTML

HTML tersusun dari tag yang dikelilingi tanda kurung sudut < >.
Setiap tag biasanya memiliki tag pembuka dan tag penutup, misalnya:

<p>Ini paragraf.</p>

Tag <p> adalah pembuka, dan </p> adalah penutup.

Berikut contoh tag dasar yang sering digunakan:

Tag Fungsi Contoh
<h1> - <h6> Judul atau heading (dari besar ke kecil) <h1>Judul Utama</h1>
<p> Paragraf <p>Ini teks paragraf.</p>
<a> Tautan atau hyperlink <a href="https://google.com">Kunjungi Google</a>
<img> Menampilkan gambar <img src="foto.jpg" alt="Foto profil">
<ul> & <li> Daftar tidak berurutan (bullet) <ul><li>Item 1<li><ul>
<ol> & <li> Daftar berurutan (angka) <ol><li>Langkah 1<li><ol>
<div> Membagi area konten atau blok <div>Bagian konten</div>
<br> Ganti baris (tidak perlu penutup) Bagian pertama <br> Baris kedua

6. Cara Membuat File HTML Pertama

  1. Buka teks editor
  2. Ketik kode HTML seperti contoh berikut:
    3. <!DOCTYPE html>
<html lang="en">
  <head>
    <title>Website Pertamaku</title>
  </head>
  <body>
    <h1>Selamat Datang di Website Saya!</h1>
    <p>Ini adalah halaman pertama saya belajar HTML.</p>
  </body>
</html>
  3. Simpan dengan nama index.html
  4. Buka file tersebut di browser. Kamu akan melihat tulisan “Selamat Datang di Website Saya!” di layar.

7. Tips Menulis Kode HTML yang Baik

  • Gunakan indentasi (tab/spasi) agar kode lebih mudah dibaca.
  • Pastikan setiap tag pembuka memiliki tag penutup yang sesuai.
  • Gunakan nama file yang jelas dan tanpa spasi.
  • Gunakan komentar untuk memberi catatan di dalam kode:
    • <!-- Ini komentar, tidak akan tampil di browser -->
  • Simpan semua file (HTML, gambar, CSS) dalam satu folder agar mudah diatur.

8. Latihan

Buat halaman web sederhana dengan ketentuan berikut:

  • Judul halaman: Profil Saya
  • Heading utama: “Selamat Datang di Website Saya”
  • Tambahkan satu paragraf deskripsi tentang dirimu.
  • Sisipkan gambar profil menggunakan tag <img>
  • Tambahkan link ke media sosial favoritmu dengan tag <a>
